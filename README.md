# House-Price-Prediction-Model
A prediction project using the Ames Housing dataset to predict house sale prices and compare important predictors like neighborhood, overall quality, and ground living area.
# House Price Prediction Model

## Tools Used
- R
- Quarto
- ggplot2
- tidyverse
- tidymodels
- AmesHousing

## What I Did
I cleaned the data, split it into training and testing data, built different linear regression models, and compared how well they predicted actual house prices.

## Findings
The model worked pretty well overall. Neighborhood, overall quality, and ground living area were the strongest predictors. The model was less accurate for higher-priced houses.
